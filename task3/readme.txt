Можно добавить к функциям, на которые больше всего подозрений в плане производительности(или вообще всем) декоратор типа:

def timing(f):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = f(*args, **kwargs)
        timing_log.info('Executed in %s' % (time.time() - start))

Создав перед этим новый логгер timing_log с форматом типа:

"[%(filename)s:%(lineno)s - %(funcName)20s() ] %(message)s"

Еще можно в форматтер добавить id тредов и процессов, если используется больше одного.
Кроме времени исполнения можно записывать еще аргументы функций, но тут надо смотреть на чувствительность данных,
которые мы пишем в лог.

После этого у нас будет лог со временем испольнения. Можно его анализировать и искать узкие места.
